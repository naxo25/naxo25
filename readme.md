### Git
    
#### Push an existing repository from the command line

    git remote add origin git@github.com:naxo25/repo.git
    git branch -M main
    git push -u origin main

#### Create a new repository on the command line

    echo "# repo" >> README.md
    git init
    git add README.md
    git commit -m "first commit"
    git branch -M main
    git remote add origin git@github.com:naxo25/repo.git
    git push -u origin main
